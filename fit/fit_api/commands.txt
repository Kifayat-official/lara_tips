1. Install latest version of nodejs
2. Nodejs comes with npx to execute npm packages without the need to install them globally.
3. npx typeorm init --name project-name-here --database database-name-here
	=> typeorm command scaffolds typescript-based typeorm project.
4. npm i -D ts-node-dev
	=> ts-node is used to run TypeScript code directly without first compiling them to JavaScript, while ts-node-dev adds hot-reloading functionality to ts-node.
	=> nodemon can also be used to auto-restart server upon changes in *.ts files.
5. npm i express
6. npm i -D @types/express

Lessons:
1. After installing typescript you can use tsc (stands for TypeScript Compiler)
2. tsc --init creates tsconfig.json
3. ts-node runs typescript in nodejs environment without precompilation.
